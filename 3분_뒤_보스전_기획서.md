## 2. 핵심 컨셉

19세 생일 자정, 주인공은 낯선 공간으로 끌려온다. 마왕은 말한다 — "너의 생일을 만끽해라… 3분간."

주인공은 3분 안에 5개의 스토리 조각(키 아이템)을 찾아야 한다. 실패하면 **이길 수 없는 보스**가 등장한다. 1분간 도망치지만, 어떻게든 끝난다.

게임플레이는 **바보같고 웃기지만**, 스토리는 **암담하고 찝찝하다.** 이 톤 대비가 게임의 정체성이다.

**핵심 키워드:** 톤 대비 · 3단계 진실 · 인과의 원 · 코즈믹 호러

## 3. 세계관

### 인과의 원

약 3000년 전, 한 청년이 타인을 구하다 죽는다. 1년 뒤 부활한다. 그가 구한 아이의 가문은 그를 "악마"로 규정하고, 대대로 처형하는 임무를 자처한다.

그의 부활 원인은 3000년 뒤 미래에서 발생할 사고의 여파다. 가문은 3000년간 그를 감금하고 연구한다. 가문의 최종 후손(주인공)이 그 연구를 계승하여, 마왕의 불사 메커니즘에서 **시간 제어 기술**을 완성한다. 그 기술의 사고가 과거로 전파되어 마왕을 만든다.

**마왕의 몸을 연구해서 만든 기술이, 마왕을 만든 원인이다.** 완전한 부트스트랩 패러독스.

### 3분의 기원

3000년간 매년 부활 즉시 처형당하던 마왕에게, 어느 해 당주가 처음으로 말한다 — "또 살아났네. 야, 힘들지? 3분만 쉬어." 벌레를 보는 듯한 무감각한 태도. 그 하찮은 한마디에 마왕의 정신이 부서진다.

### 마왕의 목적

주인공이 시간 기술을 개발할 **"마음 자체"**를 꺾는 것. 자기 가문의 죄를 목격하게 하여, 시간에 개입하려는 의지를 무너뜨린다.

### 주인공

가문의 후손. 과학자. 마왕의 불사 메커니즘을 연구하여 시간 제어 기술을 완성한 장본인. 그 사실을 아직 모른다.

## 4. 게임 플로우

### 로고 → 인트로 → 메인 메뉴

게임 시작 시 로고 → 메인 메뉴 (게임 시작, 설정 등)

### 오프닝 컷씬 (비주얼 노벨 스타일)

주인공 A(19세)의 생일 전날. 친구들과 노는 평범한 밤. 혼자 화장실에 간다. 시계가 0시를 가리키는 순간, 공간이 뒤틀리며 낯선 건물로 끌려온다.

### 튜토리얼 (스테이지 0)

건물 내부. "올라와"라는 소리가 들린다. 옥상으로 올라가며 이동 → 점프 → 대시 → 점프+대시를 순서대로 학습.

### 옥상 (마왕 조우)

옥상에 어떤 존재가 서 있다.

**마왕:** *"생일을 축하한다."*
**마왕:** *"너의 생일을 만끽해라... 3분간."*

### 스테이지 시작

스테이지 선택 후, 타이머가 0:00 → 3:00으로 빠르게 카운트업되는 애니메이션이 재생된 뒤 게임 시작.

---

### 노멀 루트 — "가문" (5스테이지, 고정맵)

대대로 마왕을 죽이는 가문의 역사. 비도덕적이고 잔혹하지만 직접 언급 없이 담담하게 서술.

| 스테이지 | 시대 | 제목 | 내용 |
|----------|------|------|------|
| 1 | 고대 | 부패한 귀족 | 농민 학대 |
| 2 | 중세 | 암살 의뢰 | 암살 청부업 |
| 3 | 근대 | 가업 계승 | 아이에게 살인 교육 |
| 4 | 현대 | 생체 실험 | 비인간적 연구 |
| 5 | 미래 | 감금 시설 | 영구 감금 + 에너지 착취 |

### 엔딩 A (노멀 완료)

화장실로 돌아옴 → 거울에 비친 것은 **마왕의 얼굴** → 화면 뒤틀림 → 레드 루트 시작.

레드에서 게임오버 되어도 레드 유지. 레드가 본편.

---

### 레드 루트 — "한 남자" (5스테이지, 룰렛 선택)

마왕이 되어 살아간 한 남자의 인생과 절망. 각 스테이지에 5개 프리셋이 있고, 시작 시 룰렛으로 선택.

| 스테이지 | 제목 | 내용 |
|----------|------|------|
| 1 | 희망의 탄생 | 평범한 아이의 탄생 |
| 2 | 희생과 죽음 | 19세, 타인을 구하다 사망 |
| 3 | 감금 시작 | 부활 후 묶여서 깨어남 |
| 4 | 반복되는 처형 | 3000년간 끝없는 죽음 |
| 5 | 탈출과 각성 | 탈출, 그리고 정신 붕괴 |

### 엔딩 B (레드 첫 클리어)

**마왕:** *"그래도 만들겠느냐? 그 기술을."*
**주인공:** *"...만들어야 하지 않나?"*

가짜 스탭롤 → 깨짐 → **"...아직 모르는구나."**

스테이지 선택 화면으로 돌아오면, 남은 4개 프리셋이 **하나로 합쳐져 기괴한 형태**가 되어 있다.

---

### 진 레드 루트 — "시간" (5스테이지, 고정 합체맵)

시간 간섭을 익힌 마왕이 과거부터 미래까지 복구하려는 노력의 과정. 전부 실패한다. 4개 시대가 뒤섞인 기괴한 맵. 룰렛 없이 고정.

| 스테이지 | 내용 |
|----------|------|
| 1~5 | 마왕의 시간 간섭 시도와 실패. 맵은 4프리셋이 합쳐진 기괴한 형태 |

### 엔딩 C (진엔딩 — 진 레드 완료)

**마왕:** *"그래도 만들겠느냐? 그 기술을."*
**주인공:** (침묵)

마왕 소멸 → *"...그래. 그게 답이지."* → 게임 세계 무너짐 → *"고마웠다."* → **게임이 꺼진다.**

재실행 시 → 평범한 생일 장면 → 아무 일도 일어나지 않음 → 조작 불가.

## 5. 게임 구조

### 기본 게임 루프

```
스테이지 진입 → 3분 카운트다운 → 아이템 탐색 → 실패 시 보스 등장 → 사망 → 반복
```

### 스테이지 구성

5개 스테이지가 1루프를 구성합니다. 노멀 루프를 완료하면 레드 루프가 해금됩니다.

### 클리어 조건

각 스테이지에 배치된 5개의 키 아이템을 3분 안에 모두 수집하면 스테이지를 클리어합니다. 5개를 모두 찾지 못한 채 시간이 만료되면, 이길 수 없는 보스가 등장하여 플레이어는 반드시 사망하고 다시 시작합니다.

## 7. 시스템

### 타이머

180초 카운트다운. 화면 상단에 항상 표시. 스테이지 시작 시 0:00→3:00 카운트업 애니메이션 후 게임 시작. 시간이 줄어들수록 시각적/청각적 연출 강화. 종료 시 보스 등장.

### 아이템

각 스테이지에 총 5개의 키 아이템이 배치. 수집 시 아이템 이름이 짧게 팝업. 스토리 조각 역할.

### 생명력

- 기본 **3개**
- 보스전에서 사망 시 **-1**
- 스테이지 내 낙사/피격은 시간 -10초 패널티 (생명력 소모 없음)
- 생명력 0 = **게임 오버** → 스테이지 1부터 재시작 (튜토리얼 생략)

### 보스전

- 3분 내 클리어 실패 시 **BOSS 연출** → 안내 문구 → 탭하면 보스전 시작

#### 보스전 첫 시작 안내 문구

> 🩸 보스전 돌입
>
> 보스전 전용 체력 🩸 3개
> 공격에 맞으면 🩸 -1
> 🩸 모두 소진 시 생명력 ❤ -1
> 1분간 생존 시 생명력 ❤ 감소 없음
>
> [탭하여 시작]

※ 첫 보스전에서만 표시. 이후에는 BOSS 연출 후 바로 탭하여 시작.
- **1분간 도망** — 난이도는 **못 깰 수준** (클리어 불가 설계)
- 보스전 전용 체력 🩸🩸🩸 (3개, 매 보스전마다 리셋)
  - 보스 공격 맞으면 🩸 -1
  - 🩸 0이면 보스전 즉사 → **기본 생명력 ❤ -1**
- 1분 버티면 → **기본 생명력 감소 없이** 스테이지 선택으로
- 기본 생명력 ❤ 0이면 게임 오버

### 보스 디자인

- **톤: 코즈믹 호러**
- 크기/모양이 **스테이지마다 다름**
- **노멀 1이 가장 크고**, 진행할수록 점점 작아짐
- **진레드 5에서는 플레이어와 동일 크기, 동일 모습**
- 보스 = 주인공 자신이라는 것을 비주얼로 암시

| 구간 | 보스 크기 | 모양 |
|------|----------|------|
| 노멀 1 | 거대 (화면의 절반 이상) | 형체를 알 수 없는 코즈믹 호러 |
| 노멀 2~3 | 대형 | 조금씩 윤곽이 보이기 시작 |
| 노멀 4~5 | 중형 | 인간형에 가까워짐 |
| 레드 1~3 | 중소형 | 인간 실루엣 |
| 레드 4~5 | 소형 | 주인공과 비슷한 체형 |
| 진레드 1~4 | 점점 동일해짐 | 주인공과 거의 같은 모습 |
| 진레드 5 | **플레이어와 동일 크기** | **같은 19살 청년. 머리 모양만 살짝 다름** |

### 클리어 플로우

3분 내 아이템 5개 수집 → 공간 뒤틀림 → 컷씬/스토리 → 스테이지 선택 화면

### 엔딩 A (노멀 완료)

노멀 5스테이지 클리어 → 컷씬: 화장실로 돌아옴 → 거울에 비친 것은 마왕의 얼굴 → 화면 뒤틀림 → 레드 루트 해금

## 8. 적 & 방해 요소

### 피격 시스템

피격/낙사 시 **시간 -10초 패널티 + 스폰 지점 리스폰**. 체력 개념 없이 시간이 곧 생명.

### 적 종류

| 적 | 행동 | 등장 |
|-----|------|------|
| **전진형** | 한 방향으로 걸어감 | N2~ |
| **왕복형** | 플랫폼 끝과 끝 사이 반복 이동 | N2~ |
| **점프 방해형** | 플레이어 점프 시 같이 뛰어서 방해 | N3~ |
| **원거리 사격형** | 고정 위치에서 투사체 발사 | N4~ |
| **추적형 (거대)** | 화면 뒤에서 쫓아오는 거대한 존재 (화면 전체 판정) | N5 |

### 방해 요소 (환경)

| 요소 | 효과 | 등장 |
|------|------|------|
| **낙석** | 특정 위치 지나가면 위에서 떨어짐 | N3~ |
| **솟아나는 가시** | 주기적으로 땅에서 솟아남 | N3~ |
| **페이크 플랫폼** | 밟으면 잠시 후 사라짐 | N4~ |
| **좌우 반전 구역** | 밟으면 조작 반전 (바보같은 플레이 유발) | N4~ |
| **감속 구역** | 이동속도 반감 | N4~ |
| **대시 봉인 구역** | 대시 사용 불가 | N5 |
| **시간 왜곡** | 타이머 2배속 감소 (레드 루트) | R1~ |
| **마왕의 손** | 화면에 거대한 손이 나와 플레이어 튕겨냄 (레드 루트) | R3~ |

### 스테이지별 적 배치 방향

- **N1**: 적 없음 (순수 탐색)
- **N2**: 전진형 + 왕복형 (기본 회피 학습)
- **N3**: 점프 방해형 + 낙석 + 솟아나는 가시
- **N4**: 원거리 사격 + 페이크 플랫폼 + 좌우 반전 구역
- **N5**: 대시 봉인 + 추적형 (최종 도전)

## 10. 아트 스타일 & 에셋 제작

### 아트 스타일

**실루엣/그림자 스타일** (LIMBO 참고)
- 캐릭터, 적, 보스, 플랫폼: 검은 실루엣
- 배경: 시대별 역광 배경 (안개, 빛줄기 등)
- 아이템만 유일하게 색이 있음
  - 노멀: 밝은 금색 (눈에 굉장히 띄게)
  - 레드: 바랜 금색 (빛이 살짝 빠진)
  - 진레드: 바랜 금색 + 핏자국

### 에셋 제작 파이프라인

| 에셋 | 도구 | 비고 |
|------|------|------|
| 인게임 스프라이트 | **Aseprite** | 실루엣이라 단색, 형태만 잡으면 됨 |
| 컷씬 이미지 (25~30장) | **OpenAI API (DALL-E)** | 프롬프트 템플릿 일괄 생성 |
| 컷씬 보정/대체 | **ChatGPT / NovelAI** | 수동 보정 또는 재생성 |
| 배경 일러스트 (6종) | **OpenAI API / ChatGPT** | 시대별 역광 배경 |
| UI | **Aseprite** 또는 코드 생성 | 미니멀 |

### 컷씬 AI 일괄 생성 계획

OpenAI API를 사용하여 프롬프트 템플릿 + 시대별 변수로 컷씬 이미지를 일괄 생성한다. 스크립트로 프롬프트 목록을 넣으면 자동으로 돌리는 방식.

- 공통 프롬프트: "dark silhouette style, backlit, atmospheric, 2D game cutscene"
- 변수: 시대(고대/중세/근대/현대/미래), 장면 설명, 분위기
