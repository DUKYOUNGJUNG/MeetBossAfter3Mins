## 2. 핵심 컨셉

플레이어는 **각기 다른 시대의 인물**로서 3분 안에 5개의 키 아이템을 찾아야 합니다.

시간이 종료되면 마왕이 등장하며, 이 보스는 **절대 이길 수 없습니다.** 플레이어는 강제 사망하고, 다시 처음부터 반복합니다.

반복 플레이를 통해 **스토리 조각을 수집**하는 것이 게임의 본질입니다.

**핵심 키워드:** 시간 제한의 긴장감 · 반복을 통한 발견 · 플레이어 주도 스토리텔링

## 3. 세계관

### 마왕의 기원

약 3000년 전, 죽어도 1년 뒤 부활하는 인간이 존재했습니다. 사람들에게 "악마"로 몰린 그에게는 특정 가문이 대대로 살해하는 임무를 부여받았습니다.

### 반복된 고통

매번 3분의 시간을 주고 처형당하는 삶이 수천 년간 반복되었습니다. 생체 실험, 고문, 감금이 이어졌습니다.

### 각성

미래 시대에 도달한 그는 과학 기술을 습득하고, 시간 간섭 능력을 획득합니다.

### 현재

마왕이 된 그는 과거 가문의 후손 5명을 각기 다른 시간대에서 끌어와, 자신이 당했던 것과 동일한 **"3분 게임"**을 강요합니다.

## 4. 게임 플로우

### 로고 → 인트로 → 메인 메뉴

게임 시작 시 로고 → 메인 메뉴 (게임 시작, 설정 등)

### 오프닝 컷씬 (비주얼 노벨 스타일)

주인공 A(19세)의 생일 전날. 친구들과 노는 평범한 밤. 혼자 화장실에 간다. 시계가 0시를 가리키는 순간, 공간이 뒤틀리며 낯선 건물로 끌려온다.

### 튜토리얼 (스테이지 0)

건물 내부. "올라와"라는 소리가 들린다. 옥상으로 올라가며 이동 → 점프 → 대시 → 점프+대시를 순서대로 학습.

### 옥상 (마왕 조우)

옥상에 어떤 존재가 서 있다.

**마왕:** *"생일을 축하한다."*
**마왕:** *"너의 생일을 만끽해라... 3분간."*

### 스테이지 시작

스테이지 선택 후, 타이머가 0:00 → 3:00으로 빠르게 카운트업되는 애니메이션이 재생된 뒤 게임 시작.

## 5. 게임 구조

### 기본 게임 루프

```
스테이지 진입 → 3분 카운트다운 → 아이템 탐색 → 실패 시 보스 등장 → 사망 → 반복
```

### 스테이지 구성

5개 스테이지가 1루프를 구성합니다. 노멀 루프를 완료하면 레드 루프가 해금됩니다.

### 클리어 조건

각 스테이지에 배치된 5개의 키 아이템을 3분 안에 모두 수집하면 스테이지를 클리어합니다. 5개를 모두 찾지 못한 채 시간이 만료되면, 이길 수 없는 보스가 등장하여 플레이어는 반드시 사망하고 다시 시작합니다.

## 6. 플레이 시스템

### 이동

2D 플랫포머 기반으로 점프, 낙하, 벽 구조를 활용합니다.

### 탐색

숨겨진 아이템 위치를 찾기 위해 맵 구조를 기억하는 것이 핵심입니다.

### 긴장감

시간 압박에 의한 긴장감이 핵심이며, 후반부에는 가속 연출이 적용됩니다.

| 조작 | 설명 |
|------|------|
| 이동 | 좌/우 방향키로 2D 평면 이동 |
| 점프 | 점프 버튼으로 플랫폼 이동 및 높은 지형 탐색 |
| 낙하 | 중력에 의한 자연스러운 낙하 및 벽 구조 활용 |
| 상호작용 | 키 아이템 및 스토리 오브젝트와 접촉하여 수집/확인 |

## 7. 시스템

### 타이머

180초 카운트다운. 화면 상단에 항상 표시되며, 시간이 줄어들수록 시각적/청각적 연출이 강화됩니다. 종료 시 즉시 보스가 등장합니다.

### 아이템

각 스테이지에 총 5개의 키 아이템이 배치됩니다. 아이템은 단순한 수집 요소가 아니라 스토리 조각 역할을 합니다. 수집 시 짧은 연출과 함께 스토리가 공개됩니다.

## 8. 적 & 방해 요소

### 피격 시스템

피격/낙사 시 **시간 -10초 패널티 + 스폰 지점 리스폰**. 체력 개념 없이 시간이 곧 생명.

### 적 종류

| 적 | 행동 | 등장 |
|-----|------|------|
| **전진형** | 한 방향으로 걸어감 | N2~ |
| **왕복형** | 플랫폼 끝과 끝 사이 반복 이동 | N2~ |
| **점프 방해형** | 플레이어 점프 시 같이 뛰어서 방해 | N3~ |
| **원거리 사격형** | 고정 위치에서 투사체 발사 | N4~ |
| **추적형 (거대)** | 화면 뒤에서 쫓아오는 거대한 존재 (화면 전체 판정) | N5 |

### 방해 요소 (환경)

| 요소 | 효과 | 등장 |
|------|------|------|
| **낙석** | 특정 위치 지나가면 위에서 떨어짐 | N3~ |
| **솟아나는 가시** | 주기적으로 땅에서 솟아남 | N3~ |
| **페이크 플랫폼** | 밟으면 잠시 후 사라짐 | N4~ |
| **좌우 반전 구역** | 밟으면 조작 반전 (바보같은 플레이 유발) | N4~ |
| **감속 구역** | 이동속도 반감 | N4~ |
| **대시 봉인 구역** | 대시 사용 불가 | N5 |
| **시간 왜곡** | 타이머 2배속 감소 (레드 루트) | R1~ |
| **마왕의 손** | 화면에 거대한 손이 나와 플레이어 튕겨냄 (레드 루트) | R3~ |

### 스테이지별 적 배치 방향

- **N1**: 적 없음 (순수 탐색)
- **N2**: 전진형 + 왕복형 (기본 회피 학습)
- **N3**: 점프 방해형 + 낙석 + 솟아나는 가시
- **N4**: 원거리 사격 + 페이크 플랫폼 + 좌우 반전 구역
- **N5**: 대시 봉인 + 추적형 (최종 도전)
